API de Alunos - Documentação

OpenAPI é uma API que permite gerenciar as informações dos alunos.

POST /alunos

Cria um novo aluno.

Parâmetro Tipo Descrição
id String ID do aluno a ser atualizado
matricula String Número de matrícula do aluno
nome String Nome completo do aluno
turma String Turma do aluno
curso String Curso em que o aluno está matriculado
turno String Turno do aluno
sexo String Sexo do aluno
nascimento Date Data de nascimento do aluno
nacionalidade String Nacionalidade do aluno
naturalidade String Naturalidade do aluno
religiao String Religião do aluno
telefone String Telefone de contato do aluno
email String Email do aluno
identidade String Número da identidade do aluno
cpf String CPF do aluno
rua String Rua do endereço do aluno
numero String Número da residência do aluno
complemento String Complemento do endereço do aluno
bairro String Bairro do aluno
cidade String Cidade do aluno
estado String Estado do aluno
cep String CEP do endereço do aluno
colegioOrigem.nome String Nome do colégio de origem do aluno
colegioOrigem.cidade String Cidade do colégio de origem
colegioOrigem.estado String Estado do colégio de origem
colegioOrigem.telefone String Telefone do colégio de origem
problemasSaude String Problemas de saúde do aluno
remediosProibidos String Remédios proibidos para o aluno
planoSaude String Plano de saúde do aluno
telefoneEmergencia String Telefone de emergência para saúde
procedimentosEmergencia String Procedimentos de emergência para o aluno
tipoSangue String Tipo sanguíneo do aluno
fatorRh String Fator Rh do tipo sanguíneo do aluno
nomeMedico String Nome do médico do aluno
telefoneMedico String Telefone do médico do aluno
nome String Nome do pai do aluno
vivo Boolean Se o pai do aluno está vivo
trabalho.local String Local de trabalho do pai
trabalho.endereco String Endereço de trabalho do pai
trabalho.profissao String Profissão do pai
trabalho.graduacao String Graduação do pai
trabalho.telefone String Telefone do trabalho do pai
dadosMae.nome String Nome da mãe do aluno
dadosMae.viva Boolean Se a mãe do aluno está viva
dadosMae.trabalho.local String Local de trabalho da mãe
dadosMae.trabalho.endereco String Endereço de trabalho da mãe
dadosMae.trabalho.profissao String Profissão da mãe
dadosMae.trabalho.graduacao String Graduação da mãe
dadosMae.trabalho.telefone String Telefone do trabalho da mãe
nome String Nome do responsável pelo aluno
cpf String CPF do responsável pelo aluno
trabalho.local String Local de trabalho do responsável
trabalho.endereco String Endereço de trabalho do responsável
trabalho.profissao String Profissão do responsável
trabalho.graduacao String Graduação do responsável
trabalho.orgao String Órgão onde o responsável trabalha
trabalho.uf String Unidade federativa do local de trabalho do responsável
trabalho.telefoneTrabalho String Telefone de trabalho do responsável
trabalho.ramal String Ramal do telefone de trabalho do responsável
rua String Rua do endereço residencial do responsável
numero String Número da residência do responsável
complemento String Complemento do endereço do responsável
bairro String Bairro do endereço residencial do responsável
cidade String Cidade do endereço residencial do responsável
estado String Estado do endereço residencial do responsável
cep String CEP do endereço residencial do responsável
telefone String Telefone residencial do responsável
celular String Celular do responsável

GET /alunos

Lista todos os alunos.

GET /alunos/:id

Obtém os detalhes de um aluno específico pelo ID.
